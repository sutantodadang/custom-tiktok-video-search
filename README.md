## Description

Custom Tiktok Video Search


## Project setup

```bash
go version 1.23
node version 20
```

## Compile and run the project

- Docker
```bash
docker compose up -d

```
- Locally
```bash
# frontend
$ cd frontend
$ pnpm dev

# backend
$ cd backend
$ go run main.go

```
